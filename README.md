  <h1>오늘 일기</h1>
  오늘 하루를 돌아보면서 이모티콘으로 표현하기, 어떤 하루를 보냈는지, 오늘 좋았던 점 혹은 아쉬웠던 점 작성하기, 좋았던 일을 다시 작성하거나, 아쉬웠던 점을 반복하지 않기 위해 개선할 수 있는 것을 작성하기 (아쉬웠던 일을 반복하기 않기 위해서 할 수 있는 노력을 작성해보세요! )
